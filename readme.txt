== Description ==

Create your own custom Admin login page with custom redirect and google captcha with our plugin, Easy to use no coding skills required for customization.</br>
Create beautiful custom login page style with your client logo & background and make your login page unique.</br>

== Installation ==

1.  Upload `Custom Admin Login | WPZest` folder to the `/wp-content/plugins/` directory
2.  Activate the plugin through the 'Plugins' menu in WordPress
3.  The plugin settings can be accessed via the 'Settings' menu in the administration area.
